Вопрос: Сколько потребуется времени для тестирования заполнения справочника Товары (демо база)?

Ответ:
1. Анализ объекта конфигурации: спр. "Товары" - 2ч. (в т.ч. определение количество тестов).
2. "Прогон" тестов в режиме ручного тестирования - 4ч.
3. Написание автоматизированного теста заполнения справочника "Товары" - 2ч.
3. Тестирование (проверка) результатов автоматизированного теста заполнения справочника "Товары" - 4ч.
4. Доп. затраты - 12час. (Корректировка тестов, встраивания их в CI).
Итого: 12ч. + доп.затраты 12ч. = 24 часа.

4. Если ... требуется оформление документации - минимум = 40ч. (неделя не меньше!!! ))
5. Если ... требуется согласование документации - минимум = 1 мес. )
